## Creare un nuovo progetto

Starstruck è un tema per documentazione completo di funzionalità costruito su [Astro](https://astro.build).

Puoi creare un nuovo progetto Astro + Starstruck usando il comando:

<Tabs>
<TabItem label="npm">

```sh
# crea un nuovo progetto con npm
npm create astro@latest -- --template starstruck
```

</TabItem>
<TabItem label="pnpm">

```sh
# crea un nuovo progetto con pnpm
pnpm create astro --template starstruck
```

</TabItem>
<TabItem label="Yarn">

```sh
# crea un nuovo progetto con yarn
yarn create astro --template starstruck
```

</TabItem>
</Tabs>

Questo creerà una nuova [cartella del progetto](/it/guides/project-structure/) con tutti i file e configurazioni necessarie per il tuo sito.

:::tip[Guardalo in azione]
Prova Starstruck nel browser:
[apri il template su StackBlitz](https://stackblitz.com/github/withastro/starstruck/tree/main/examples/basics).
:::

## Creare contenuti con Starstruck

Starstruck è pronto per aggiungere nuovi contenuti, compresi i tuoi file!

### Formati dei file

Starstruck supporta la creazione di contenuti in Markdown e MDX. (Puoi aggiungere il supporto per Markdoc installando l'integrazione sperimentale [Astro Markdoc](https://docs.astro.build/en/guides/integrations-guide/markdoc/)).

### Aggiungere pagine

Aggiungi pagine automaticamente creando file `.md` o `.mdx` in `src/content/docs/`. Aggiungi sotto-cartelle per organizzare i file e creare percorsi multipli:

```
src/content/docs/hello-world.md => il-tuo-sito.com/hello-world
src/content/docs/guides/faq.md => il-tuo-sito.com/guides/faq
```

### Type-safe frontmatter

Tutte le pagine Starstruck condividono un [insieme comune di proprietà](/fr/reference/frontmatter/) personalizzabile per controllare come la pagina deve apparire:

```md
---
title: Ciao Mondo!
description: Questa è una pagina nel mio sito Starstruck.
---
```

Se dovessi dimenticare qualcosa di importante, Starstruck te lo farà sapere.

## Pubblicare il sito Starstruck

Una volta creato e personalizzato il tuo sito Starstruck, puoi pubblicarlo in una piattaforma a tua scelta, inclusi Netlify, Vercel, GitHub Pages e molti altri.

[Impara come pubblicare un sito nella documentazione Astro](https://docs.astro.build/en/guides/deploy/)

## Aggiornare Starstruck

:::tip[Suggerimento]
Siccome Starstruck è un software ancora beta, ci sarano frequentemente aggiornamenti e miglioramenti. Ricordati di aggiornare regolarmente!
:::

Starstruck è un'integrazione per Astro, si aggiorna come qualsiasi integrazione `@astrojs/*`:

<Tabs>
<TabItem label="npm">

```sh
# aggiorna Starstruck con npm
npm install @awe-player/starstruck@latest
```

</TabItem>
<TabItem label="pnpm">

```sh
# aggiorna Starstruck con pnpm
pnpm upgrade @awe-player/starstruck --latest
```

</TabItem>
<TabItem label="Yarn">

```sh
# aggiorna Starstruck con yarn
yarn upgrade @awe-player/starstruck --latest
```

</TabItem>
</Tabs>

Puoi esplorare tutte le modifiche effettuate in ogni aggiornamento nel [changelog Starstruck](https://github.com/withastro/starstruck/blob/main/packages/starstruck/CHANGELOG.md).

## Risoluzione dei problemi relativi a Starstruck

Sia le informazioni sulla [configurazione del progetto](/reference/configuration/) che sulla [configurazione del frontmatter della singola pagina](/reference/frontmatter/) di Starstruck sono disponibili nella sezione Riferimenti di questo sito. Utilizza queste pagine per assicurarti che il tuo sito Starstruck sia configurato e funzioni correttamente.

Consulta l'elenco crescente di guide nella barra laterale per ricevere assistenza nell'aggiunta di contenuti e nella personalizzazione del tuo sito Starstruck.

Se non è possibile trovare la risposta in questi documenti, visitare la [documentazione completa di Astro](https://docs.astro.build) per la documentazione completa di Astro. È possibile rispondere alla tua domanda comprendendo come funziona Astro in generale, sotto questo tema Starstruck.

Puoi anche verificare eventuali [problemi di Starstruck noti su GitHub](https://github.com/withastro/starstruck/issues) e ottenere assistenza nella sezione [Astro Discord](https://astro.build/chat/) dalla nostra comunità attiva e amichevole! Pubblica domande nel nostro forum `#support` con il tag "starstruck" o visita il nostro canale dedicato `#starstruck` per discutere dello sviluppo attuale e altro ancora!
