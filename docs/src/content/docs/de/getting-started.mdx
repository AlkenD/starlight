## Erstelle ein neues Projekt

Starstruck ist ein voll ausgestattetes Dokumentations-Theme, das auf dem [Astro](https://astro.build) Framework aufbaut.

Du kannst ein neues Astro + Starstruck Projekt mit dem folgenden Befehl erstellen:

<Tabs>
<TabItem label="npm">

```sh
# Erstelle ein neues Projekt mit npm
npm create astro@latest -- --template starstruck
```

</TabItem>
<TabItem label="pnpm">

```sh
# Erstelle ein neues Projekt mit pnpm
pnpm create astro --template starstruck
```

</TabItem>
<TabItem label="Yarn">

```sh
# Erstelle ein neues Projekt mit yarn
yarn create astro --template starstruck
```

</TabItem>
</Tabs>

Damit wird ein neues [Projektverzeichnis](/de/guides/project-structure/) mit allen erforderlichen Dateien und Konfigurationen für deine Webseite erstellt.

:::tip[In Aktion sehen]
Probiere Starstruck in deinem Browser aus:
[öffne die Vorlage in StackBlitz](https://stackblitz.com/github/withastro/starstruck/tree/main/examples/basics).
:::

## Inhalte mit Starstruck erstellen

Starstruck ist bereit für dich, neuen Inhalt hinzuzufügen oder deine vorhandenen Dateien mitzubringen!

### Dateiformate

Starstruck unterstützt das Erstellen von Inhalten in Markdown und MDX. (Du kannst die experimentelle [Astro Markdoc Integration](https://docs.astro.build/de/guides/integrations-guide/markdoc/) installieren, um Markdoc zu unterstützen.)

### Seiten hinzufügen

Füge neue Seiten zu deiner Webseite automatisch hinzu, indem du `.md` oder `.mdx` Dateien in `src/content/docs/` erstellst. Erstelle Unterordner, um deine Dateien zu organisieren und mehrere Pfadsegmente zu erstellen:

```
src/content/docs/hello-world.md => your-site.com/hello-world
src/content/docs/guides/faq.md => your-site.com/guides/faq
```

### Typsichere Frontmatter

Alle Starstruck Seiten teilen sich anpassbare [Frontmatter-Eigenschaften](/de/reference/frontmatter/), mit denen das Erscheinungsbild der Seite gesteuert wird:

```md
---
title: Hello, World!
description: This is a page in my Starstruck-powered site
---
```

Wenn du etwas Wichtiges vergisst, wird Starstruck dich daran erinnern.

## Veröffentlichung deiner Starstruck Webseite

Sobald du deine Starstruck Webseite erstellt und angepasst hast, kannst du sie auf einen Webserver oder Hosting-Plattform deiner Wahl veröffentlichen, einschließlich Netlify, Vercel, GitHub Pages und vielen mehr.

[Lerne mehr über die Veröffentlichung einer Astro Webseite in der Astro Dokumentation.](https://docs.astro.build/de/guides/deploy/)

## Starstruck aktualisieren

:::tip
Da es sich bei Starstruck um eine Beta-Software handelt, wird es regelmäßig Updates und Verbesserungen geben. Achte darauf Starstruck regelmäßig zu aktualisieren!
:::

Starstruck ist eine Astro-Integration und wird wie jede `@astrojs/*`-Integration aktualisiert:

<Tabs>
<TabItem label="npm">

```sh
# Starstruck mit npm aktualisieren
npm install @awe-player/starstruck@latest
```

</TabItem>
<TabItem label="pnpm">

```sh
# Starstruck mit pnpm aktualisieren
pnpm upgrade @awe-player/starstruck --latest
```

</TabItem>
<TabItem label="Yarn">

```sh
# Starstruck mit yarn aktualisieren
yarn upgrade @awe-player/starstruck --latest
```

</TabItem>
</Tabs>

Eine vollständige Liste der Änderungen findest du im [Starstruck Changelog](https://github.com/withastro/starstruck/blob/main/packages/starstruck/CHANGELOG.md).

## Fehlerbehebung

Sowohl die Starstruck [Projektkonfiguration](/reference/configuration/) als auch die [Konfiguration einzelner Seiten](/reference/frontmatter/) findest du im Referenzbereich dieser Website. Nutze diese Seiten, um sicherzustellen, dass deine Starstruck-Site richtig konfiguriert ist und funktioniert.

In der wachsenden Liste der Anleitungen in der Seitenleiste findest du Hilfe beim Hinzufügen von Inhalten und beim Anpassen deiner Starstruck-Website.

Wenn du keine Antwort in dieser Dokumentation finden kannst, besuche bitte die [Astro Docs](https://docs.astro.build) für die vollständige Astro-Dokumentation. Deine Frage kann vielleicht beantwortet werden, wenn du verstehst, wie Astro im Allgemeinen funktioniert.

Du kannst auch im [Astro Discord](https://astro.build/chat/) Hilfe von unserer aktiven, freundlichen Community bekommen. Du kannst Fragen in unserem `#Support`-Forum stellen oder unseren speziellen `#starstruck`-Channel besuchen, um aktuelle Entwicklungen und mehr zu diskutieren.
