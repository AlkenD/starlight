## Créer un nouveau projet

Starstruck est un thème de documentation complet construit sur le framework [Astro](https://astro.build).

Vous pouvez créer un nouveau projet Astro + Starstruck en utilisant la commande suivante :

<Tabs>
<TabItem label="npm">

```sh
# créer un nouveau projet avec npm
npm create astro@latest -- --template starstruck
```

</TabItem>
<TabItem label="pnpm">

```sh
# créer un nouveau projet avec pnpm
pnpm create astro --template starstruck
```

</TabItem>
<TabItem label="Yarn">

```sh
# créer un nouveau projet avec yarn
yarn create astro --template starstruck
```

</TabItem>
</Tabs>

Cela créera un nouveau [répertoire de projet](/fr/guides/project-structure/) avec tous les fichiers et configurations nécessaires pour votre site.

:::tip[Voir en action]
Essayez Starstruck dans votre navigateur :
[ouvrir le modèle sur StackBlitz](https://stackblitz.com/github/withastro/starstruck/tree/main/examples/basics).
:::

## Créer du contenu avec Starstruck

Starstruck est conçu pour que vous puissiez ajouter du nouveau contenu, ou apporter vos fichiers existants !

### Formats de fichiers

Starstruck prend en charge la création de contenu en Markdown et MDX. (Vous pouvez ajouter la prise en charge de Markdoc en installant l'intégration expérimentale [Astro Markdoc integration](https://docs.astro.build/fr/guides/integrations-guide/markdoc/).)

### Ajouter des pages

Ajoutez automatiquement de nouvelles pages à votre site en créant des fichiers `.md` ou `.mdx` dans `src/content/docs/`. Ajoutez des sous-dossiers pour organiser vos fichiers, et pour créer plusieurs segments de chemin :

```
src/content/docs/hello-world.md => votre-site.com/hello-world
src/content/docs/guides/faq.md => votre-site.com/guides/faq
```

### Type-safe frontmatter

Toutes les pages Starstruck partagent un [ensemble commun de propriétés de présentation](/fr/reference/frontmatter/) personnalisable pour contrôler l'apparence de la page :

```md
---
title: Bonjour, le monde !
description: Ceci est une page de mon site web propulsé par Starstruck.
---
```

Si vous oubliez quelque chose d'important, Starstruck vous le fera savoir.

## Déployer votre site web Starstruck

Une fois que vous avez créé et personnalisé votre site Web Starstruck, vous pouvez le déployer sur un serveur Web ou une plateforme d'hébergement de votre choix, y compris Netlify, Vercel, GitHub Pages et bien d'autres.

[Pour en savoir plus sur le déploiement d'un site Astro, consultez la documentation Astro](https://docs.astro.build/fr/guides/deploy/)

## Mettre à jour Starstruck

:::tip
Comme Starstruck est un logiciel bêta, il y aura des mises à jour et des améliorations fréquentes. Assurez-vous de mettre à jour Starstruck régulièrement !
:::

Starstruck est une intégration Astro, et est mis à jour comme toute intégration `@astrojs/*` :

<Tabs>
<TabItem label="npm">

```sh
# mettre à jour Starstruck avec npm
npm install @awe-player/starstruck@latest
```

</TabItem>
<TabItem label="pnpm">

```sh
# mettre à jour Starstruck avec pnpm
pnpm upgrade @awe-player/starstruck --latest
```

</TabItem>
<TabItem label="Yarn">

```sh
# mettre à jour Starstruck avec yarn
yarn upgrade @awe-player/starstruck --latest
```

</TabItem>
</Tabs>

Vous pouvez voir une liste complète des modifications apportées à chaque version dans [le journal des modifications de Starstruck](https://github.com/withastro/starstruck/blob/main/packages/starstruck/CHANGELOG.md).
