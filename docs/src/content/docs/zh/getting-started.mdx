## 创建一个新项目

Starstruck 是一个基于 [Astro](https://astro.build) 框架构建的全功能文档主题。

你可以使用以下命令创建一个新的 Astro + Starstruck 项目：

<Tabs>
<TabItem label="npm">

```sh
# 使用 npm 创建一个新项目
npm create astro@latest -- --template starstruck
```

</TabItem>
<TabItem label="pnpm">

```sh
# 使用 pnpm 创建一个新项目
pnpm create astro --template starstruck
```

</TabItem>
<TabItem label="Yarn">

```sh
# 使用 yarn 创建一个新项目
yarn create astro --template starstruck
```

</TabItem>
</Tabs>

这将创建一个新的 [项目目录](/zh/guides/project-structure/)，其中包含你网站所需的所有文件和配置。

:::tip[查看实际效果]
在浏览器中尝试 Starstruck：
[在 StackBlitz 上打开模板](https://stackblitz.com/github/withastro/starstruck/tree/main/examples/basics)。
:::

## 使用 Starstruck 创建内容

Starstruck 已经准备好让你添加新的内容，或者将你现有的文件带过来！

### 文件格式

Starstruck 支持使用 Markdown 和 MDX 来编写内容。（你可以通过安装实验性的 [Astro Markdoc 集成](https://docs.astro.build/en/guides/integrations-guide/markdoc/) 来添加对 Markdoc 的支持。）

### 添加页面

在 `src/content/docs/` 中创建 `.md` 或 `.mdx` 文件，就可以自动将新页面添加到你的站点中。添加子文件夹来组织你的文件，并创建多个路径段：

```
src/content/docs/hello-world.md => your-site.com/hello-world
src/content/docs/guides/faq.md => your-site.com/guides/faq
```

### 类型安全的 frontmatter

所有 Starstruck 页面都共享一个可自定义的 [frontmatter 属性集](/zh/reference/frontmatter/)，用于控制页面的外观：

```md
---
title: Hello, World!
description: This is a page in my Starstruck-powered site
---
```

如果你忘记了任何重要的东西，Starstruck 会提醒你。

## 部署你的 Starstruck 网站

一旦你创建并自定义了你的 Starstruck 网站，你就可以将它部署到你选择的 web 服务器或托管平台上，包括 Netlify、Vercel、GitHub Pages 等等。

[在 Astro 文档中了解如何部署 Astro 网站。](https://docs.astro.build/zh-cn/guides/deploy/)

## 更新 Starstruck

:::tip[提示]
由于 Starstruck 是 beta 软件，所以会经常更新和改进。请务必定期更新 Starstruck！
:::

Starstruck 是一个 Astro 集成，可以像任何 `@astrojs/*` 集成一样更新：

<Tabs>
<TabItem label="npm">

```sh
# 使用 npm 升级 Starstruck
npm install @awe-player/starstruck@latest
```

</TabItem>
<TabItem label="pnpm">

```sh
# 使用 pnpm 升级 Starstruck
pnpm upgrade @awe-player/starstruck --latest
```

</TabItem>
<TabItem label="Yarn">

```sh
# 使用 yarn 升级 Starstruck
yarn upgrade @awe-player/starstruck --latest
```

</TabItem>
</Tabs>

你可以在 [Starstruck 更新日志](https://github.com/withastro/starstruck/blob/main/packages/starstruck/CHANGELOG.md) 中查看每个版本的变动列表。
